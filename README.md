## Retently 介绍

Retently是一个专注于客户体验管理的SaaS平台，主打NPS（净推荐值）、CSAT（客户满意度）和CES（客户费力度）三大核心指标的自动化调研。这家公司成立于2014年，总部位于美国特拉华州，主要服务B2B SaaS企业、电商平台和订阅制业务。

它最大的特点是能跟你现有的工具无缝对接——Salesforce、HubSpot、Intercom、Stripe这些常用系统都能直接打通。系统会在合适的时机自动触发调研（比如客户完成购买后、使用产品30天后），然后把反馈数据实时同步到你的CRM里，省去了大量手动操作。

Retently的仪表盘设计挺直观的，能实时看到NPS趋势、客户分层、流失预警这些关键信息。它还有个AI驱动的情感分析功能，能自动识别客户反馈里的正面、负面情绪，帮你快速定位问题。对于需要系统化管理客户反馈、想提升续费率的公司来说，这是个挺实用的工具。

## Retently 测评

从用户反馈和第三方评测来看，Retently在客户体验管理工具里算是中上水平，特别适合需要系统化管理客户反馈的B2B公司。

**用户普遍认可的点**：

集成能力确实强，特别是跟Salesforce和HubSpot的双向同步做得很顺滑。有个用户提到他们的客户成功团队以前要手动整理NPS数据，接入Retently之后这个环节完全自动化了，每周能省下好几个小时。

自动化触发逻辑比较灵活。你可以设置"客户使用产品满30天后发送NPS调研"或者"订单完成72小时后发CSAT问卷"这种精准时机的触发，比人工记着发邮件靠谱多了。

仪表盘的可视化做得不错，NPS趋势、客户分层、流失预警这些关键指标一目了然。非技术背景的团队成员也能快速上手看懂数据。

**被吐槽的问题**：

价格对小团队来说有点贵。如果你只是想偶尔发几次问卷，用免费的Typeform或Google Forms可能更实惠。Retently更适合需要持续、系统化跟踪客户满意度的公司。

高级功能（比如自定义品牌、深度细分）要Professional套餐才有，这意味着你得至少付到$159/月。如果预算紧张，可能会觉得功能解锁门槛有点高。

邮件模板的自定义程度有限。虽然可以改颜色、logo，但布局选项不多，有些追求品牌调性的团队可能会觉得不够灵活。

**第三方平台评分**：

在G2上Retently拿到了4.5/5星的评分（基于500+条评价），主要优势体现在易用性和客户支持上。Capterra上的评分也差不多，用户特别提到客服响应速度快，遇到技术问题基本当天能解决。

总体来说，如果你的公司有明确的客户体验管理需求、预算充足、需要跟现有系统深度集成，Retently是个值得考虑的选择。但如果你只是想简单收集反馈、预算有限，可能需要先评估一下ROI是否划算。
